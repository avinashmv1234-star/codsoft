# Task 3 - Iris Classification

This directory contains a Jupyter notebook for performing classification on the classic Iris dataset.

## Files

- `iris_classification.ipynb` - Notebook demonstrating data loading, exploration, model training, and evaluation using the Iris dataset.

## Description

The notebook walks through preprocessing the Iris dataset, training a machine learning model (e.g., logistic regression or another classifier), and evaluating its performance. It provides visualizations and explanations appropriate for educational purposes.

## Requirements

The environment should have the usual data science libraries installed, such as:

- numpy
- pandas
- scikit-learn
- matplotlib
- seaborn

Alternatively, refer to `requirements.txt` in the root or other tasks for guidance.

## Usage

Open the notebook in Jupyter and run the cells sequentially to follow along with the Iris classification workflow. Feel free to modify the model or dataset as an exercise.
